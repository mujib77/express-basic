# Express Basics

A simple Express.js server built while learning backend development.

## What I learned
- Setting up an Express server
- GET and POST routes
- Middleware (express.json)
- Sending JSON responses
- In-memory data storage with arrays

## Routes
- GET `/` - Hello World
- GET `/about` - About page
- GET `/contact` - Contact info
- GET `/user` - Returns a user object
- GET `/users` - Returns all created users
- POST `/createuser` - Creates a new user

## How to run
npm install
node app.js
